Projeto prático focado na melhoria da experiência do usuário em uma plataforma de streaming. O trabalho envolve entender como os usuários utilizam o sistema hoje (AS-IS), definir uma navegação mais simples e personalizada (TO-BE), implementar as melhorias e acompanhar resultados por indicadores (KPIs), com o objetivo de facilitar a escolha de conteúdos, reduzir desistências e tornar o uso mais intuitivo no dia a dia.
